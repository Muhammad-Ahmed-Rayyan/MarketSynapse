# MarketSynapse

Real-time financial intelligence: live news → FinBERT sentiment → price correlation → LLM-generated market brief, in one pipeline.

> Educational/informational project only. Not financial advice.

## Status
🚧 Day 4 of 12 complete — news, sentiment, price correlation, and LangGraph-generated market briefs are all live.

## Quickstart

```bash
git clone <your-repo-url> marketsynapse
cd marketsynapse

python3 -m venv .venv
source .venv/bin/activate      # Windows: .venv\Scripts\activate

pip install -r requirements.txt

cp .env.example .env
# Add your NEWS_API_KEY (https://newsapi.org) and GROQ_API_KEY (https://console.groq.com)

uvicorn backend.main:app --reload
# Visit http://localhost:8000/docs for interactive Swagger UI
```

## Endpoints

| Endpoint | Description |
|---|---|
| `GET /news/{ticker}` | Recent news articles for a ticker, filtered for relevance |
| `GET /stock/{ticker}` | Price history and % change over the period |
| `GET /analyze/{ticker}` | Combined sentiment + price with an alignment verdict (aligned/diverged/mixed) |
| `GET /brief/{ticker}` | LLM-generated plain-English market brief |

All accept an optional `days_back` query param (default 7).

## Known limitations
- NewsAPI free tier: 100 requests/day, articles limited to the past month
- yfinance is an unofficial scraper of Yahoo Finance — occasional transient failures are expected, not bugs
- FinBERT is finance-domain-trained but can misread sarcasm or routine corporate language (e.g. dividend announcements) as negative
- Market briefs are generated by an LLM constrained to the provided data only — verify against source headlines before treating as fact
